# Sources and Trust Notes

Public users can now see sources directly in the UI on content pages and service detail pages. This file remains the maintainer-facing source map for common official references.

## Preferred source order

1. Government of Ontario
2. Government of Canada
3. Official Kingston-area hospitals and public health organizations
4. Official service provider websites
5. Community organizations only when they are the service owner or operator

## Core official sources used in this project

### Provincial and federal

- Ontario health system overview: `https://www.ontario.ca/page/health-care-ontario`
- OHIP application and eligibility: `https://www.ontario.ca/page/apply-ohip-and-get-health-card`
- Health Care Connect / find a family doctor or nurse practitioner: `https://www.ontario.ca/page/find-family-doctor-or-nurse-practitioner`
- Health811: `https://www.ontario.ca/page/get-medical-advice-health811`
- Interim Federal Health Program: `https://www.canada.ca/en/immigration-refugees-citizenship/services/refugees/help-within-canada/health-care/interim-federal-health-program.html`
- Ontario Health atHome: `https://ontariohealthathome.ca/`

### Kingston and regional

- Kingston Health Sciences Centre: `https://kingstonhsc.ca/`
- Kingston Community Health Centres: `https://kchc.ca/`
- KFL&A Public Health: `https://www.kflaph.ca/`
- AMHS-KFLA: `https://www.amhs-kfla.ca/`
- Providence Care: `https://www.providencecare.ca/`

## Source handling rules

- Link to the exact official page when possible, not just the homepage
- Avoid copying unsupported clinic claims from third-party directories
- Prefer date-stamped service review notes over blanket global verification claims
- Use the visible UI trust labels to match the actual confidence level
